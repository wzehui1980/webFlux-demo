📦 Mono 示例（单个对象）

GET /api/hello?key={name}
简单的字符串返回示例，支持可选参数

请求参数：

key String [可选] 默认值：哈哈
GET /api/helloMono/{name}
路径参数示例，返回 Mono

路径参数：

name String *必填 用户名
GET /api/helloMap
随机生成 Map 数据示例，每次返回不同的键值对

🌊 Flux 示例（多个对象/流）

GET /api/helloFlux
返回 Flux，包含多个字符串元素
GET /api/helloFlux2
返回 Flux，包含多个用户对象
GET /api/see
Server-Sent Events (SSE) 流式传输示例，每500ms推送一个数据

👥 用户管理 API

GET /api/users
查询所有用户列表
GET /api/users/{id}
根据ID查询单个用户

路径参数：

id Long *必填 用户ID
POST /api/users
新增用户
PUT /api/users/{id}
更新用户信息
DELETE /api/users/{id}
删除用户
GET /api/users/{id}/phone
查询用户手机号（map 转换示例）
